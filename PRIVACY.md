# Privacy Policy for Prompt Pins

**Last Updated: January 7, 2025**

## Data Collection
Prompt Pins does **NOT** collect, transmit, or share any user data.

## Data Storage
- All pinned prompts are stored locally in your browser using Firefox's built-in storage API
- Data never leaves your device
- No analytics, tracking, or telemetry of any kind
- No external network requests are made by this extension

## Permissions Explained
The extension requests the following permissions:

### `storage`
Used to save your pinned prompts locally in your browser so they persist across sessions.

### `contextMenus`
Used to add the "Pin prompt" option when you right-click on selected text.

### `*://chat.openai.com/*` and `*://chatgpt.com/*`
Required to:
- Display the sidebar on ChatGPT pages
- Read selected text when you create a pin
- Fill the ChatGPT input box when you use a pin
- Auto-submit prompts when requested

The extension only runs on ChatGPT domains and does not access any other websites.

## Third Parties
This extension does not share data with any third parties because it does not collect any data.

## Your Data Rights
Since all data is stored locally in your browser:
- You can delete all pins at any time by clearing your browser data
- Uninstalling the extension removes all stored pins
- Your data is never backed up or synchronized outside your browser

## Source Code
This extension is open source. You can review the complete source code to verify these privacy claims.

## Contact
If you have questions about this privacy policy, please use the support options available on the Firefox Add-ons listing page.

Alternatively, submit feedback through the Firefox Add-ons review system.

## Changes to This Policy
Any changes to this privacy policy will be reflected in the extension's version notes and this document.
